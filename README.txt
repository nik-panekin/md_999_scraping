Written with Python 3.13.0

This script performs data scraping from the https://999.md/ website. It
retrieves essential information for all products within a specified category.

The results are saved in both JSON and CSV formats.

To run the script:
    python md_999_scraper.py

To remove previously saved results and restart the scraping process, use
the --restart flag:
    python md_999_scraper.py --restart

Feel free to use and modify this script as needed.

Refer to requirements.txt for dependencies.
